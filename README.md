# AB – Paiements

Gestion des factures fournisseurs pour AB Renov 35

## Features

- 📊 Tableau de bord avec KPIs
- 📋 Gestion des factures
- 💰 Suivi des paiements
- 📅 Alertes échéances
- ☁️ Sync Google Sheets

## Installation

```bash
npm install
npm start
```

## Déploiement

```bash
npm run deploy
```
